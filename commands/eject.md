# /bottle:eject

Leave bottle management and manage tools manually.

## Usage

```bash
bottle eject
```

## What it does

1. Sets mode to "ejected"
2. Keeps all tools and plugins in place
3. Stops bottle from managing updates

After ejecting, you're responsible for updating tools manually.

## If bottle is not installed

First install the bottle CLI:

```bash
cargo install bottle
```
