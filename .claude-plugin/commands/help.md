# /bottle:help

Show available bottle commands.

## Usage

```
/bottle:help
```

## Response

Display the following to the user:

```
Bottle - Open Horizon Labs Tool Stack

START HERE:
  /bottle:getting-started    Learn how to use the tools
  /bottle:workflow           Get guided workflow for your task

DAILY USE:
  /dive-prep                 Prepare session context (start here!)
  /ba:status                 See your tasks
  /superego:review           Get feedback before committing

SETUP:
  /bottle:init               Initialize all tools with defaults
  /bottle:status             Show installed tools and versions
  /bottle:install [name]     Install a bottle (default: stable)

MANAGEMENT:
  /bottle:update             Update to latest bottle snapshot
  /bottle:switch <name>      Switch to a different bottle
  /bottle:list               List available bottles
  /bottle:integrate          Add/remove platform integrations
  /bottle:eject              Stop bottle management (keep tools)

More info: https://github.com/open-horizon-labs/bottle
```
